scoreboard

Scoreboard is an application designed as an information radiator. It uses visual and audio feedback for maximum effect. It is designed to support multiple continuous integration servers. The application currently includes connectors for Hudson and Cruise Control.

This version of scoreboard represents a new development direction for the original leapstream developed scoreboard.
